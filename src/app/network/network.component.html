Network Security Basics

Course Description

This course covers the basics of securing networks. Students will explore firewalls, encryption, VPNs, and other tools used to protect network infrastructure. By the end of the course, students will be able to apply fundamental network security practices to safeguard data and communications.

Learning Outcomes

  • Understand the importance of network security in the modern digital world
  • Learn about firewalls, VPNs, encryption, and other network security tools
  • Gain practical knowledge of securing personal and organizational networks
  • Be prepared for more advanced network security courses

Prerequisites

Introduction to Cybersecurity or basic networking knowledge

Duration & Schedule

  • Total Duration: 1 Month
  • Offline (Saturday & Sunday): 10:00 AM - 3:00 PM (5 hours each day)
  • Online (Monday to Thursday): 8:00 PM - 9:00 PM (1 hour each day)
  • Total Course Hours: 40 hours (20 offline + 20 online)
