Ethical Hacking Essentials

Course Description

In this intermediate course, students will learn the ethical hacker's mindset and various tools used for penetration testing. From vulnerability assessments to exploiting weaknesses, this course provides the skills needed to test and improve system security ethically.

Learning Outcomes

  • Understand the ethical hacking process and methodologies
  • Gain hands-on experience using tools like Kali Linux and Metasploit
  • Learn how to conduct vulnerability assessments and penetration tests
  • Develop a strong foundation for advanced penetration testing techniques

Prerequisites

Introduction to Cybersecurity & Network Security Basics

Duration & Schedule

  • Total Duration: 1 Month
  • Offline (Saturday & Sunday): 10:00 AM - 3:00 PM (5 hours each day)
  • Online (Monday to Thursday): 8:00 PM - 9:00 PM (1 hour each day)
  • Total Course Hours: 40 hours (20 offline + 20 online)
