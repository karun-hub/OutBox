Cryptography Essentials

Course Description

This course explores the essential techniques of cryptography used to secure communication and protect data. Learn how encryption, hashing, and digital signatures work, and how cryptography plays a critical role in modern security protocols.

Learning Outcomes

  • Understand how cryptography is used to secure digital communication
  • Learn about encryption algorithms, hashing, and key management
  • Be able to implement basic cryptographic solutions for security
  • Prepare for specialized cryptography topics in advanced courses

Prerequisites

Introduction to Cybersecurity, Network Security Basics

Duration & Schedule

  • Total Duration: 1 Month
  • Offline (Saturday & Sunday): 10:00 AM - 3:00 PM (5 hours each day)
  • Online (Monday to Thursday): 8:00 PM - 9:00 PM (1 hour each day)
  • Total Course Hours: 40 hours (20 offline + 20 online)
